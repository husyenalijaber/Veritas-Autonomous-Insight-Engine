## 1. Core Concept & Value: The "Trust But Verify" Agent

**Veritas** is an autonomous research engine built using the **Google Gemini API** to solve the critical problem of unreliable information generated by standard LLMs. In fields like **Climate Change** and **Public Health**, unverified facts pose significant risks. Veritas mandates a rigorous, structured verification process by implementing the **Orchestrator Pattern**.

### Key Value Proposition:
* **Automated Verification:** Significantly reduces the need for manual human fact-checking time.
* **Hallucination Reduction:** Separation of the data gathering (Researcher) from data validation (**Analyst**) eliminates inconsistencies and un verified claims in the final report.
* **Scalable Impact:** Provides verifiable and objective insights, ideal for policy-relevant research.
